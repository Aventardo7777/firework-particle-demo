# WebGL 高性能烟花粒子渲染特效
### 🎆 在线演示地址：https://Aventardo7777.github.io/firework-particle-demo/

## 项目概述
基于原生WebGL硬件加速实现的烟花粒子视觉特效，优化粒子销毁、复用、色彩渐变逻辑，大幅降低渲染性能损耗，适配PC浏览器实时播放烟花爆炸、升空、消散全套动态效果。

## 核心特性
- ✨ 粒子对象池复用，减少频繁创建销毁带来的性能开销
- 🎆 多层烟花爆炸逻辑，分层色彩渐变、轨迹随机偏移
- 🚀 原生JS无框架依赖，单HTML文件开箱即用
- 🎨 动态重力、风速、生命周期参数可调
- 📌 轻量化渲染，低配置设备也能稳定高帧率运行

## 本地运行
1. 克隆仓库
git clone https://github.com/Aventardo7777/firework-particle-demo.git

2. 使用 Chrome/Edge 主流浏览器直接打开项目主文件 firework-particle-optimized.html
3. 无需额外环境配置，打开即可查看完整烟花动态效果
